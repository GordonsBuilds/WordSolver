# Wordhunt Solver

Static Vite app for solving a 4x4 Wordhunt board.

## How it works

- Enter one letter in each square of the 4x4 grid.
- Click `Solve board` to find every valid word formed by adjacent letters.
- Letters can be used in any direction, but the same tile cannot be reused in one word.
- Paste your own dictionary into the text area if you want to search a custom word list.

## Scripts

- `npm run dev` - start the local Vite dev server
- `npm run build` - create a production build for GitHub Pages
- `npm run preview` - preview the production build

## GitHub Pages

The app builds with a relative Vite base, so it can be hosted from a project page without changing asset paths.
